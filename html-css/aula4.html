Adicionando CSS na brincadeira

3 formas de utilizar o CSS:

  • Configurar o estilo na próprioa definição da tag.
    Não é indicado pois só funciona para aquele elemento definido e de difícil manutenção
  • Adicionar dentro da tag head a definição de estilos para cada tag que se deseje customizar. Não é indicado pois seria necessário repetir essas mesmas configurações para as outras páginas html do site.
  • Adicionar todas as customizações em um outro arquivo e linkar essas configuração na tag head.
    Assim existe a vantagem do reaproveitamento.
